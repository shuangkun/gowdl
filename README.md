# gowdl

A Workflow Description Language (WDL) parser in Go.


## Getting started

### Install from source

To install `gowdl` from source, first clone this repo:

```
https://github.com/w-gao/gowdl.git
```

Then, generate the necessary parsers:

```
make generate
```

Finally, install the local package:

```
go install .
```
